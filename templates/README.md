# Excel 템플릿 가이드

이 폴더에는 Excel 자동화 앱에서 사용할 수 있는 템플릿 파일들이 저장됩니다.

## 기본 템플릿 구조

### Rawdata 시트 예시

Excel 파일은 반드시 "Rawdata"라는 이름의 시트를 포함해야 합니다.

| 최지현  | 개발팀   | 26   | 4000000  | 2022-01-10  | 사원   |

## 템플릿 작성 규칙

### 필수 요구사항

1. **시트 이름**: 반드시 "Rawdata"여야 합니다 (대소문자 정확히 일치)
2. **헤더 행**: 첫 번째 행은 컬럼 헤더(제목)여야 합니다
3. **데이터 행**: 두 번째 행부터 실제 데이터가 시작됩니다

### 권장사항

1. **컬럼 이름**: 의미 있고 명확한 이름 사용
2. **데이터 타입**: 각 컬럼의 데이터 타입을 일관되게 유지
3. **빈 셀**: 가능한 빈 셀을 최소화
4. **날짜 형식**: 일관된 날짜 형식 사용 (예: YYYY-MM-DD)
5. **숫자 형식**: 천 단위 구분 기호 사용 가능 (자동으로 숫자로 인식됨)

### 주의사항

1. **병합된 셀**: 사용하지 않는 것이 좋습니다
2. **빈 행/열**: 데이터 중간에 빈 행이나 열이 없어야 합니다
3. **수식**: 수식이 아닌 값으로 저장되어야 합니다
4. **매크로**: 매크로는 처리되지 않습니다

## 생성되는 보고서 내용

애플리케이션은 Rawdata 시트를 분석하여 다음과 같은 보고서를 자동 생성합니다:

### 1. 기본 통계

각 컬럼에 대해:
- 총 데이터 수
- 고유값 수
- 빈값 수

### 2. 숫자 컬럼 분석

숫자 데이터가 있는 컬럼에 대해:
- 최솟값
- 최댓값
- 평균
- 합계

### 3. 빈도 분석

모든 컬럼에 대해:
- 상위 5개 빈도값 및 출현 횟수

## 템플릿 파일 만들기

### Excel에서 직접 만들기

1. Microsoft Excel 실행
2. 새 통합 문서 생성
3. 시트 이름을 "Rawdata"로 변경
4. 첫 행에 컬럼 헤더 입력
5. 두 번째 행부터 데이터 입력
6. `.xlsx` 형식으로 저장

### Google Sheets에서 만들기

1. Google Sheets에서 스프레드시트 생성
2. 시트 이름을 "Rawdata"로 변경
3. 데이터 입력
4. `파일 > 다운로드 > Microsoft Excel (.xlsx)` 선택

### LibreOffice Calc에서 만들기

1. LibreOffice Calc 실행
2. 시트 이름을 "Rawdata"로 변경
3. 데이터 입력
4. `.xlsx` 형식으로 저장

## 대용량 데이터 처리 팁

### 성능 최적화

- **데이터 정리**: 불필요한 컬럼 제거
- **데이터 분할**: 10만 행 이상의 데이터는 분할 처리 권장
- **메모리 관리**: 처리 전 다른 브라우저 탭 닫기

### 권장 데이터 크기

- **최적**: 1,000 ~ 10,000 행
- **양호**: 10,000 ~ 50,000 행
- **느릴 수 있음**: 50,000 ~ 100,000 행
- **권장하지 않음**: 100,000 행 이상 (분할 처리 권장)

## 문제 해결

### "Rawdata 시트를 찾을 수 없습니다"

- 시트 이름이 정확히 "Rawdata"인지 확인 (띄어쓰기, 대소문자 주의)
- 다른 시트가 아닌 "Rawdata" 시트에 데이터가 있는지 확인

### "데이터가 없습니다"

- Rawdata 시트가 비어있지 않은지 확인
- 첫 행 외에 최소 1개 이상의 데이터 행이 있는지 확인
- 빈 행으로 시작하지 않는지 확인

### 일부 데이터가 누락됨

- 병합된 셀이 있으면 데이터가 제대로 읽히지 않을 수 있음
- 빈 행이나 열이 중간에 있으면 데이터가 잘릴 수 있음
- 모든 셀의 병합을 해제하고 빈 행/열을 제거
