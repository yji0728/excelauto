## 각 샘플의 예상 보고서

### 직원 데이터 보고서

**통계:**
- 총 직원: 10명
- 평균 연령: 30.5세
- 평균 급여: 5,070,000원
- 부서별 인원: 개발팀(5), 영업팀(3), 마케팅팀(2)
- 성비: 남(6), 여(4)

**빈도 분석:**
- 가장 많은 부서: 개발팀
- 가장 많은 직급: 대리(4명)
- 급여 범위: 4,000,000 ~ 6,000,000원

### 판매 데이터 보고서

**통계:**
- 총 거래: 15건
- 총 판매액: 22,260원
- 평균 거래액: 1,484원
- 가장 많이 팔린 제품: 노트북A
- 가장 많은 판매 지역: 서울

**빈도 분석:**
- 카테고리: 악세서리(9), 전자제품(6)
- 최다 판매자: 김철수
- 제품별 판매량 Top 3: 마우스C, 키보드B, 노트북A

### 설문조사 보고서

**통계:**
- 총 응답: 15명
- 평균 만족도: 3.87
- 추천 의향: 예(10), 아니오(4), 보통(1)
- 응답률이 높은 연령대: 20-30대

**빈도 분석:**
- 가장 많은 연령대: 20대, 30대
- 지역별 분포: 서울(5), 기타(10)
- 사용기간: 1년(5), 6개월(4)

## 대량 데이터 생성 팁

### Excel 수식 사용

더 많은 행을 빠르게 생성하려면:

1. **날짜 자동 증가:**
   ```
   A2: 2024-01-01
   A3: =A2+1
   (A3을 아래로 드래그)
   ```

2. **랜덤 숫자:**
   ```
   =RANDBETWEEN(최솟값, 최댓값)
   예: =RANDBETWEEN(20, 50)  // 20~50 사이 랜덤 숫자
   ```

3. **목록에서 랜덤 선택:**
   ```
   =CHOOSE(RANDBETWEEN(1,3), "서울", "부산", "대구")
   ```

4. **순차 번호:**
   ```
   A2: 1
   A3: =A2+1
   ```

### 수식을 값으로 변환

수식으로 데이터를 생성한 후:
1. 모든 데이터 선택
2. 복사 (Ctrl+C)
3. 선택 영역에 마우스 우클릭
4. "선택하여 붙여넣기" > "값"
5. 확인

이렇게 하면 수식이 값으로 변환되어 안정적으로 작동합니다.

## 주의사항

1. **시트 이름**: 반드시 "Rawdata"여야 함
2. **헤더**: 첫 행은 컬럼 이름
3. **데이터 타입**: 각 컬럼은 일관된 데이터 타입 유지
4. **빈 셀**: 최소화
5. **병합 셀**: 사용하지 않기

## 테스트 시나리오

### 기본 테스트
1. 10행 정도의 작은 데이터로 시작
2. 모든 기능이 정상 작동하는지 확인
3. 보고서가 올바르게 생성되는지 확인

### 데이터 타입 테스트
1. 숫자 컬럼만 있는 데이터
2. 텍스트 컬럼만 있는 데이터
3. 혼합된 데이터
4. 날짜 컬럼이 있는 데이터

### 대용량 테스트
1. 1,000행
2. 10,000행
3. 50,000행 (성능 확인)

### 에러 테스트
1. 빈 Rawdata 시트
2. 잘못된 시트 이름
3. 헤더만 있고 데이터 없음
4. 중간에 빈 행이 있는 데이터

## 저장 위치

생성한 샘플 파일을:
- `templates/` 폴더에 저장하면 참조용으로 사용 가능
- 단, `.gitignore`에 의해 Git에는 커밋되지 않음
- 필요시 `example_*.xlsx` 이름으로 저장하면 Git에 포함됨
